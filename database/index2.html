B. Perintah DML (Data Manipulation Language)

DML adalah sub perintah dari bahasa SQL yang digunakan untuk memanipulasi data dalam database yang telah dibuat. Terdapat empat perintah penting dalam DML, yaitu INSERT, SELECT, UPDATE, dan DELETE.

Perintah-perintah pada DDL (Data Manipulation Language) :

1. SELECT

Perintah yang digunakan untuk menampilkan data dari satu atau lebih tabel dalam sebuah basis data yang sama.

Contoh :

SELECT nama_kolom1, nama_kolom2 FROM nama_tabel;


2. UPDATE

Perintah digunakan untuk mengubah atau memperbarui data yang sudah ada.

Contoh :

UPDATE nama_tabel SET kolom1=data1, kolom2=data2,… WHERE kolom=data;


3. INSERT

Perintah yang digunakan untuk menambah data baru.

Contoh :

INSERT INTO nama_tabel VALUES (data1, data2, dst…);


3. DELETE

Perintah yang digunakan untuk menghapus data.

Contoh :

DELETE FROM nama_tabel WHERE kolom=data;
