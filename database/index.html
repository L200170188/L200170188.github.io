PERINTAH SQL (Structured Query Language)




A. Perintah DDL (Data Definition Language)

DDL (Data Definition Language) merupakan kumpulan perintah SQL yang digunakan untuk membuat, mengubah dan menghapus struktur dan definisi metadata dari objek-objek database.

Perintah-perintah pada DDL (Data Definition Language) :

1. CREATE

Perintah ini digunakan untuk membuat, termasuk di antaranya membuat database baru, tabel baru, view baru, dan kolom.

Contoh :

CREATE DATABASE nama_database;

Penjelasan :

Pada contoh perintah di atas membuat sebuah tabel.


2. ALTER

Perintah ini digunakan untuk mengubah struktur tabel yang telah dibuat. Pekerjaannya mencakup mengganti nama tabel, menambah kolom, mengubah kolom, menghapus kolom, maupun memberikan atribut pada kolom.

Contoh :

ALTER TABLE nama_tabel ADD nama_kolom datatype;

Penjelasan :

Perintah contoh perintah di atas akan ditambahkan kolom baru dengan tipe data dan akan ditambahkan pada tabel yang akan dipilih


3. DROP

Perintah ini digunakan untuk menghapus database dan tabel.

Contoh :

DROP DATABASE nama_database;

Penjelasan :

Pada contoh perintah di atas menghapus sebuah database.
